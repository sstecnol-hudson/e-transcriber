                    TROUBLESHOOTING RÁPIDO
================================================================================

❌ Erro CORS?
  → Use servidor local (http://localhost:8000)
  → Não use file://

❌ Protocolos não carregam?
  → Limpe cache (Ctrl+Shift+Delete)
  → Limpe localStorage (DevTools → Application → Clear All)

❌ Questionário não aparece?
  → Verifique console (F12)
  → Procure por mensagens de erro (❌)

❌ Arquivo não encontrado?
  → Verifique: modules/qualification/data/protocol-*.json
  → Deve ter 3 arquivos JSON
